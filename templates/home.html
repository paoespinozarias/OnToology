What is OnToology

A system to automate part of the collaborative ontology development process. Given a repository with an owl file, OnToology will survey it and produce diagrams, a complete documentation and validation based on common pitfalls.

OnToology can handle OWL and RDFS vocabularies in RDF+XML and Turtle serialization.

How to use it

1. Enter your repo as user/repo i.e. myuser/myrepo (it does not work with organizations at the moment).

2. You will be redirected to Github to authorize OnToology to access your repo.

3. Update your ontology and push.

4. OnToology will create pull request including generated files (documentation, diagrams and evaluation)

5. Merge the pull request created by OnToology

What's new

New interface: We changed the interface to a bright theme with a better representation

Faster login: We moved the extra step for private repositories detection to be done automatically.

Bundle Download: Now you can download your ontology bundle of documentation, diagrams and evaluation report with a single click.

Publish with w3id: You can publish persistent permanent URIs with w3id.org using OnToology.

Previsualization with vocabLite: A nice and a clear way to visualize your ontologies in your repositories.
